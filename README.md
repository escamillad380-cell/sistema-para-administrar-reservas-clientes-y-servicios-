# Sistema de Gestión - Software FJ

## Descripción
Este proyecto tiene como objetivo desarrollar un sistema de gestión de clientes, servicios y reservas que sea integral. Está construido en Python sin el uso de bases de datos, empleando programación orientada a objetos.

## Objetivo
Crear una aplicación sólida utilizando los principios de encapsulación, herencia, polimorfismo y abstracción, además de gestionar excepciones.
## Funcionalidades
- Registro de clientes
- Creación de servicios
- Gestión de reservas
- Manejo de errores con excepciones
- Registro de eventos en logs
